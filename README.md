# ACM-Godot-SIG
A set of examples used for the ACM GameDev SIG made in Godot<br>

## Ex5-3D-Controller-and-Static-Methods
In this meeting we talked about the basics of developing games in 3D.<br>
For starters, we talked about how rendering works in 3D with the use of the WorldEnvironment node and light sources. Following that we built a basic 3D environment using CSG nodes. We also built a 3D character and camera controller with support for both keyboard and mouse, and controller. The implementation of the character controller is similar to the one we made for 2D, just modified to fit the extra dimension. We then built a modular composition based hitbox hurtbox and health system that we can use to create our entities and projectiles. Finally we briefly talked about static functions which we used to bundle all the initialization and instantiation logic of our bullet scene to the Bullet3D class itself.
